### 项目介绍<br>
该项目主要用于订餐，餐品加入购物车，显示商家信息，显示商品详情页，显示评价页面
<br>

### 运行该项目的命令<br>
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
